# Проект по обработке естественного языка (NER)

Этот проект посвящен извлечению именованных сущностей (Named Entity Recognition, NER) с использованием различных моделей машинного обучения.

## Структура проекта

*   `data/`: Содержит исходные (`raw/`) и обработанные (`processed/`) данные.
*   `models/`: Хранит обученные модели NER.
*   `notebooks/`: Jupyter-ноутбуки для экспериментов, предобработки данных и реализации моделей.
*   `scripts/`: Исполняемые Python-скрипты для анализа ошибок, сравнения моделей и других задач.
*   `outputs/`: Выходные данные, результаты экспериментов и логи.
